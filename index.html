How it works
1
Drop in the skill
Copy a folder into .agents/skills/. Takes 30 seconds.
2
Run /unkode
Your agent analyzes the codebase and writes unkode.yaml + a Mermaid diagram.
3
Open a PR
The GitHub Action posts a color-coded architecture diff as a comment.
